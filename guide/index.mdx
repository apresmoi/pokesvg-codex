# PokeSVG

PokeSVG is a Pokedex-style web app where every creature is generated as SVG from a deterministic genome.

## Quick start

- Press `Discover` to generate a new encounter.
- Press `A` to **Catch** the encounter, or `B` to **Let Go**.
- In the dex list, use `Up/Down` to move selection.
- Press `A` (or `Right`) to open the selected mon (detail view).
- Press `B` (or `Left`) to go back to the list.
- Press `Export` in detail view to copy a share code to your clipboard.
- Paste (`Ctrl+V` / `Cmd+V`) anywhere in the app to import a share code (or JSON).
- Press `System Config` to change presets, background, and animations.
