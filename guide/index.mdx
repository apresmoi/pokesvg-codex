# PokeSVG

PokeSVG is a Pokedex-style web app where every creature is generated as SVG from a deterministic genome (JSON).

## Quick start

- Press `DISCOVER` to generate a new mon.
- Use the D-pad to move selection in the list.
- Press `A` to open the selected mon (detail view).
- Press `B` to go back to the list.
- Press `EXPORT` in detail view to copy the genome JSON to your clipboard.
- Paste (`Ctrl+V`) anywhere in the app to import a genome JSON back into your dex.
- Press `CFG` to open System Config (presets, background, animations).
