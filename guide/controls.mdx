# Controls

PokeSVG is meant to feel like a device. All controls are inside the SVG Pokedex.

## Buttons

- `LIST`: go to the dex list screen
- `CFG`: go to System Config
- `DISCOVER`: generate a new mon and select it
- `A`: confirm / open / toggle (depends on the screen)
- `B`: back
- D-pad `UP` / `DOWN`: move selection (list) or move the highlighted option (System Config)

## Screens

### Dex List

- D-pad moves selection.
- `A` opens the selected mon (detail view).

### Dex Detail

- `EXPORT` copies the selected genome JSON.
- Paste (`Ctrl+V`) imports a genome JSON.

### System Config

- D-pad moves the highlighted option.
- `A` toggles / cycles the selected option.
- `B` returns to the list.
